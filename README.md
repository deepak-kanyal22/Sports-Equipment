# Sports-Equipment

A **responsive sports equipment website** showcasing a variety of sports products with dedicated pages for browsing items, checking size guides, reading about the project, and contacting the creators.

---

## Link to the Website:-
- https://deepak-kanyal22.github.io/Sports-Equipment/

## Features

- **Static landing page** highlighting major sports categories (e.g., cricket, football, badminton, gym gear).
- **Responsive layout** using pure HTML and CSS, ensuring the site works well on desktops, tablets, and mobile screens.
- **Simple and intuitive navigation** with sections such as:
  - Hero banner  
  - Product cards  
  - About section  
  - Contact section / Footer  
- Clean and minimal design suitable for a beginner-friendly or academic project.

---

## Tech Stack

- **HTML5** – Structure, content, and layout of the pages.  
- **CSS3** – Styling, colours, typography, spacing, and responsiveness.

---

## Getting Started

1. **Clone the repository**

   ```bash
   git clone https://github.com/Ashish2106B/Sports-Equipment.git
   ```
2. Open the project folder in any code editor (e.g., VS Code).
3. Run the project
- Either open index.html directly in your browser or Use a live server extension (recommended for smoother development).

## Project Structure

| File              | Description                                                             |
| ----------------- | ----------------------------------------------------------------------- |
| `index.html`      | Main landing page with the overall website layout.                      |
| `products.html`   | Displays available sports items or categories.                          |
| `about.html`      | Contains information about the project or team.                         |
| `contact.html`    | Provides a contact section or form.                                     |
| `size-guide.html` | Offers sizing information for different sports equipment.               |
| `style.css`       | Main stylesheet controlling layout, colours, fonts, and responsiveness. |
